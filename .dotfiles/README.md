
## Structure

.
├── .zshrc            - Sourced zsh files, including functions, definitions, and options
├── config
│   ├── ohmyzsh
│   └── tabs
├── hooks             - Scripts that run automatically at certain events
│   ├── linux
│   └── macOS
├── logs              - Log files
├── scripts
│   ├── lang          - Scripts categorized by programming language
│   ├── macOS
│   ├── path          - Scripts that are added to PATH when sourcing the .zshrc
│   └── secret
├── setup             - Setup scripts to setup a machine of the corresponding OS
│   ├── linux
│   ├── macOS
│   └── windows
├── src               - Source code files for compiled languages
└── tmp               - Temporary files, used by scripts for intermediate processing
